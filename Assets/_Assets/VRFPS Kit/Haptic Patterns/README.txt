This package supports VR Haptic Patterns out of the box!

It allows you to feel complex vibration patterns propagating
through objects, like the individual clicks and vibrations 
as you e.g. slowly pull a trigger or rack a pistol slide.

In order to install it you simply import the package

------------------------------------
VR Haptic Patterns is available on the Asset store:
https://assetstore.unity.com/packages/tools/input-management/vr-haptic-patterns-248615
